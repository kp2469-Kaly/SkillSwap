# SkillSwap

**Version: 1.0.0**

A local peer-to-peer skill exchange platform that connects individuals for community-driven learning through costless skill barter sessions. SkillSwap eliminates the high barriers to entry of formal education by enabling users to share knowledge, schedule learning sessions, and build meaningful connections within their local communities.

## Features

- **Smart Matching Algorithm**: Connects users based on proximity, availability, and skill compatibility
- **Session Scheduling**: Coordinate both online and offline learning exchanges
- **Video Call Integration**: Built-in WebRTC support for remote learning sessions
- **Expert Consultation**: Access to paid expert services for specialized guidance
- **Community Groups**: Join skill-focused communities and participate in group discussions
- **Review & Rating System**: Comprehensive feedback system for quality assurance
- **Location-based Services**: Find skill partners in your local area

## Technology Stack

- **Backend**: Node js
- **Frontend**: React
- **Database**: PostgreSQL


## Usage

### Getting Started

1. **Create an Account**: Sign up with your email and create a profile
2. **Add Skills**: List your expertise and skills you want to learn
3. **Find Matches**: Browse suggested skill partners in your area
4. **Schedule Sessions**: Coordinate learning exchanges through the platform
5. **Join Communities**: Participate in skill-focused groups
6. **Leave Reviews**: Rate your learning experiences to help others
